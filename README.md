# README

* Ruby/Rails version

        ruby 3.0.3
        rails 6.1.6

* Configuration

    Just bundle install

* Database creation

        rails db:create

* Database initialization

        rails db:migrate

* Database seed

        rails db:seeds

    Tests are not implemented. I just add some generators like Faker and FactoryBot.
